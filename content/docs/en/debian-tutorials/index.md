This section collects step-by-step guides around Debian-based servers.
The tutorials target anyone who wants to set up, maintain or extend a
self-hosted server.

<Callout type="info">
The detailed tutorials below are currently only available in German.
The code blocks (all the actual commands) work the same way in any
language, of course. If you need an English version of a specific guide,
feel free to open an issue on
[GitHub](https://github.com/Musiker15/Website/issues).
</Callout>
